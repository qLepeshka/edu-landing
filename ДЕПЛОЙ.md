# 🚀 ИНСТРУКЦИЯ ПО РАЗВЁРТЫВАНИЮ ЛЕНДИНГА

## Почему лендинг не работает постоянно?

Когда вы запускаете проект через `python -m http.server` или подобную команду, сайт работает **только пока запущен терминал**. Это как лампочка — выключили выключатель, свет погас.

**Для постоянной работы** сайт нужно разместить на **хостинге** — удалённом сервере, который работает 24/7.

## 📋 ВАРИАНТЫ ПОСТОЯННОГО ХОСТИНГА

### ✅ ВАРИАНТ 1: GitHub Pages (БЕСПЛАТНО, РЕКОМЕНДУЕТСЯ)

**Плюсы:**
- ✅ Бесплатно навсегда
- ✅ Надёжно (серверы GitHub)
- ✅ Автоматические бэкапы
- ✅ Свой домен можно подключить

**Шаг 1: Регистрация на GitHub**
1. Перейдите на https://github.com
2. Нажмите "Sign up"
3. Заполните форму (email, пароль, имя пользователя)
4. Подтвердите email

**Шаг 2: Установка Git на компьютер**
1. Скачайте с https://git-scm.com/download/win
2. Установите (все настройки по умолчанию)
3. Перезапустите командную строку

**Шаг 3: Создание репозитория**
1. На GitHub нажмите "+" → "New repository"
2. Название: `edu-landing`
3. Visibility: Public
4. Нажмите "Create repository"

**Шаг 4: Загрузка файлов**
Откройте командную строку в папке лендинга:
```bash
cd C:\РАБОТА\лендинг
git init
git add .
git commit -m "Первая версия лендинга"
git branch -M main
git remote add origin https://github.com/ВАШ_НИК/edu-landing.git
git push -u origin main
```

**Шаг 5: Включение сайта**
1. На GitHub перейдите в ваш репозиторий
2. Settings → Pages (слева в меню)
3. Source: Deploy from a branch
4. Branch: main / (root)
5. Нажмите "Save"

**ГОТОВО!** Через 1-2 минуты сайт будет доступен по адресу:
```
https://ВАШ_НИК.github.io/edu-landing/
```

### ✅ ВАРИАНТ 2: Netlify (БЕСПЛАТНО, САМЫЙ ПРОСТОЙ)

**Плюсы:**
- ✅ Бесплатно
- ✅ Не нужен Git
- ✅ Просто перетащите папку

**Способ А: Через браузер (самый простой)**
1. Перейдите на https://netlify.com
2. Зарегистрируйтесь (кнопка "Sign up")
3. Войдите в аккаунт
4. Перейдите в "Sites"
5. **Перетащите папку** `лендинг` в область "Drag and drop your site output folder here"
6. Сайт сразу опубликован!

**Способ Б: Через командную строку**
```bash
# Установка Netlify CLI
npm install -g netlify-cli

# Перейдите в папку
cd C:\РАБОТА\лендинг

# Авторизация
netlify login

# Деплой
netlify deploy --prod
```

**ГОТОВО!** Сайт доступен по адресу вида:
```
https://random-name-12345.netlify.app
```

## 🔗 СВОЙ ДОМЕН (опционально)

Если хотите адрес типа `www.mysite.ru` вместо `mysite.github.io`:

1. **Купите домен** на:
   - https://reg.ru (от 199₽/год)
   - https://nic.ru

2. **Подключите к хостингу:**
   
   **GitHub Pages:**
   - Settings → Pages → Custom domain
   - Введите ваш домен
   - Следуйте инструкциям по DNS
   
   **Netlify:**
   - Domain settings → Add custom domain
   - Введите домен
   - Обновите DNS записи

## 🎯 SEO ПРОДВИЖЕНИЕ

### Чтобы сайт находили в Google и Яндекс:

**1. Зарегистрируйте в поисковиках:**
- Google: https://search.google.com/search-console
- Яндекс: https://webmaster.yandex.ru

**2. Добавьте сайт:**
- Введите URL сайта
- Подтвердите права (загрузите файл)

**3. Заполните meta-теги:**
Откройте `index.html` и измените:
```html
<title>Ваше название сайта</title>
<meta name="description" content="Описание сайта для поисковиков">
```

**4. Добавьте аналитику:**
- Google Analytics: https://analytics.google.com
- Яндекс.Метрика: https://metrika.yandex.ru

## 🆘 ЧАСТЫЕ ВОПРОСЫ

**В: Сайт будет работать вечно?**
О: Да, пока вы не удалите его с хостинга. Бесплатные тарифы не имеют срока действия.

**В: Можно ли редактировать сайт после публикации?**
О: Да! Измените файлы и загрузите заново (или сделайте новый commit в Git).

О: Бесплатные тарифы: 0₽. Свой домен: ~200₽/год.

**В: Выдержит ли сайт нагрузку?**
О: Да, бесплатные тарифы выдерживают тысячи посетителей в день.
